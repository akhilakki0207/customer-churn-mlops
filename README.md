# Customer Churn Prediction - MLOps Project

This project uses FastAPI + Scikit-learn to train a churn prediction model and deploy it using CI/CD pipelines (GitHub Actions/Jenkins), Docker, and AWS.

## Features
- FastAPI REST API
- Docker containerization
- GitHub Actions CI/CD
- MLflow experiment tracking
- Terraform-based AWS infrastructure

## To Run Locally
```bash
docker-compose up --build
